> **TL;DR**: Minecraft PvP progression is locked as six training blocks. A day is one rep inside the current block; a stage is not one day. Advancement requires passing the gate, not calendar time.

## Growth Contract
- Parent branch: [[projects/minecraft-pvp/README]]
- Node role: progression contract
- First parent link: [[projects/minecraft-pvp/README]]
- Growth trigger: split into stage-specific notes after 3+ review reports or repeated failures accumulate inside one stage.
- Forbidden contents: raw session logs, mod setup details, unverified server claims, montage tech, and advanced weapon obsession before the current gate is passed.
- Source/evidence boundary: this note defines the training architecture; fight-specific evidence belongs in review reports under [[projects/minecraft-pvp/notes/minecraft-pvp-video-review-workflow]].

## Core Rule

```text
Stage = training block
Day = one rep inside current block
Pass gate = proof you can leave the block
```

Calendar time does not advance the project. Competence advances it.

## Daily Template

For a normal 1-2 hour session:

```text
10 min   warmup / aim / movement
40-70m   main gamemode reps
10-20m   focused weakness reps
10m      review 1-2 fights or write failure label
```

For a 60-minute day:

```text
5m warmup
40m reps
10m review
5m note
```

For a 120-minute day:

```text
10m warmup
80m reps
20m review
10m note
```

Daily output:

```text
one main drill
one failure label
one next correction
```

## Stage Map

```text
1. Sword Spacing Kernel
2. Sword Fight Stability
3. Hotbar + Defensive Utility
4. SMP Utility Fighting
5. Mace / Spear Layer
6. Aerial Lancer Integration
```

Current stage:

```text
Stage 1: Sword Spacing Kernel
```

## Stage 1 - Sword Spacing Kernel

Main gamemode:

```text
sword duels / basic kit duels
```

Goal:

```text
stop turning every fight into face-to-face panic
```

Daily work:

```text
10-20 sword duels
one focus only: edge-of-range entry
no spear, no mace, no utility obsession
```

Pass gate:

```text
2 sessions in a row:
- 10+ sword fights each session
- 60%+ fights without hard W-key collapse
- opponent is not constantly filling more than 40% of screen
- Duc can name why he lost
```

Expected duration:

```text
3-7 sessions
```

## Stage 2 - Sword Fight Stability

Main gamemode:

```text
sword duels
OP/survival duels only if still mostly sword-based
```

Goal:

```text
spacing + cooldown + aim survive pressure
```

Daily work:

```text
10-15 sword duels
5 fights intentionally played slower
review one death
```

Pass gate:

```text
2 sessions in a row:
- spam-click panic is mostly gone
- Duc can recover after losing first hit
- crosshair stays in opponent body lane
- resets go to medium range, not full escape
```

Expected duration:

```text
4-10 sessions
```

## Stage 3 - Hotbar + Defensive Utility

Main gamemode:

```text
SMP kit / survival kit / shield kit
```

Goal:

```text
use tools without losing hands
```

Daily work:

```text
8-12 utility fights
practice shield/offhand, food/heal, pearl, blocks/water
only add 1-2 tools per day
```

Pass gate:

```text
2 sessions in a row:
- heal before total panic
- offhand/shield swap on purpose
- pearl or disengage without random slot collapse
- recover from wrong slot in under 2 seconds
```

Expected duration:

```text
5-12 sessions
```

## Stage 4 - SMP Utility Fighting

Main gamemode:

```text
full SMP / modern PvP kit
```

Goal:

```text
fight as a system, not as a sword player holding random items
```

Daily work:

```text
8-15 full-kit fights
one reviewed clip every session
label losses: spacing / hotbar / commit / terrain / late heal
```

Pass gate:

```text
3 sessions total, not necessarily consecutive:
- use 3+ tools in one fight intentionally
- stop full-committing from neutral
- know when to reset, chase, heal, pearl, or pressure
- most deaths have a clear label, not "idk"
```

Expected duration:

```text
1-3 weeks
```

## Stage 5 - Mace / Spear Layer

Main gamemode:

```text
mace/spear kit if available
otherwise custom/modern SMP modes with those mechanics
```

Goal:

```text
add identity weapons without corrupting fundamentals
```

Daily work:

```text
50% normal SMP fights
50% mace/spear-specific fights
review every bad mace/spear attempt
```

Pass gate:

```text
- no random mace fishing from neutral
- spear is used for reach/spacing, not panic poking
- mace is used after setup, height, or punish window
- sword rhythm survives weapon swapping
```

Expected duration:

```text
2-4 weeks
```

## Stage 6 - Aerial Lancer Integration

Main gamemode:

```text
complex modern kit
elytra/aerial/spear/mace/utility if server supports it
```

Goal:

```text
final style: spacing + utility + aerial pressure + spear/mace punish
```

Daily work:

```text
full-kit fights
intentional aerial entries
review only meaningful losses
train one system at a time: entry, punish, escape, chase
```

Pass gate:

```text
in reviewed fights:
- aerial entry has purpose
- spear/mace appears after setup
- utility supports pressure or survival
- Duc is not pressing cool buttons as compensation
```

Expected duration:

```text
open-ended main mode
```
