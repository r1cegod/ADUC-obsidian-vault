> **TL;DR**: First two weeks are not for becoming flashy; they are for locking setup, sword stability, axe/shield basics, SMP utility, and small mace reliability.

## Growth Contract
- Parent branch: [[projects/minecraft-pvp/README]]
- Node role: plan leaf
- First parent link: [[projects/minecraft-pvp/README]]
- Growth trigger: convert into weekly review notes after 14 days of actual session results.
- Forbidden contents: permanent build claims, server claims without validation, raw clip dumps, and unbounded daily journaling.
- Source/evidence boundary: this is the executable plan derived from the gathered roadmap; raw/source pointers live in [[projects/minecraft-pvp/sources/README]].

## Day 1-3

```text
Goal: remove setup chaos and start sword reps.

Setup:
- create separate PvP profile
- lock FOV/sensitivity
- choose one PvP visibility pack stack
- disable visual noise
- confirm server rules before adding risky QoL mods

Training:
- sword duels only
- no spear/mace except casual 5-minute curiosity cap
- note repeated losses
```

Pass signal:

```text
I can enter fights without feeling mechanically lost.
```

## Day 4-7

```text
Split:
70% sword
30% axe/shield

Focus:
- spacing
- cooldown rhythm
- crosshair tracking
- sprint control
- shield bait/disable timing
- stop panic-jumping
```

Pass signal:

```text
I can name why I lost most fights.
```

## Day 8-10

```text
Split:
50% sword
25% axe/shield
25% SMP utility

Focus:
- blocks
- pearls
- bow/crossbow pressure
- food/heal timing
- backing out before death
```

Pass signal:

```text
I use utility before panic, not after death is already certain.
```

## Day 11-14

```text
Split:
50% sword/axe maintenance
30% mace drills
20% SMP utility

Focus:
- 3-block and 5-block mace hits
- wind charge to mace only if available/allowed
- miss recovery
- do not drop into shields
```

Pass signal:

```text
Small mace hits are boringly reliable.
```

## Daily Note Format

Keep it tiny:

```text
date:
time trained:
main drill:
server/map:
most common death:
one fix tomorrow:
```

## Week 2 Exit Gate

Only add spear seriously after this is true:

```text
sword/axe no longer feels alien
hotbar does not collapse under pressure
small mace hits sometimes land on purpose
I can escape at least some losing fights
```
