> **TL;DR**: This hub is the default entrypoint for PathFinder evaluation work, routing from repo-level audit status to runner usage, stage-local logs, and retrieval-agent evaluation rules.

## Summary
Use this hub when the task is about attack datasets, replay execution, evaluation policy, or stage hardening status. It connects the top-level evaluation router to the runner guide, retrieval-agent rules, and the stage-specific audit logs.

This domain is already large enough that hub-based navigation is better than opening individual eval notes one by one from the README.
