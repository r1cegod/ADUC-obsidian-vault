> **TL;DR**: This hub is the default entrypoint for PathFinder architecture work, routing from high-level system shape down to the exact state and documentation contracts.

## Summary
Use this hub for anything about graph shape, state ownership, architecture contracts, or documentation structure. Start with the overview doc, then move to the state contract if the task changes fields or routing logic.

This hub exists to keep architecture reading hub-based instead of forcing agents to jump between multiple leaf notes directly. It is the architecture cluster center for the PathFinder project workspace.
