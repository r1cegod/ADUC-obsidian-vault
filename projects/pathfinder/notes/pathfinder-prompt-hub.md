> **TL;DR**: This hub is the routing page for PathFinder prompt work, covering stage-agent prompt contracts, output-compiler prompt assembly, and the broader production prompt playbook.

## Summary
Use this hub for prompt audits, prompt design questions, and any task that changes how reasoning or student-facing output is instructed. It separates the stage-agent contract from the output-compiler contract and points to the broader prompt-engineering guide only when needed.

This keeps prompt work disciplined and avoids bouncing between unrelated prompt notes.
