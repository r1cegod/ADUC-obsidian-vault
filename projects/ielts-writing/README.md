## The Gap

Band 4-5 -> 7-8 is a **schema acquisition** problem, not a language problem.
English is strong; the missing piece is IELTS-specific structure, cohesion patterns, and lexical precision.

## Docs

| # | Doc | Mutations | Status |
|---|-----|-----------|--------|
| 1 | [[projects/ielts-writing/notes/task1-academic\|Task 1 Academic]] | line/bar/pie/table/process/map (6 graph types) | done |
| 2 | [[projects/ielts-writing/notes/task2-essay\|Task 2 Essay]] | opinion(2) / discussion / adv-dis(2) / problem-solution / two-part (7 mutations) | done |

Each doc follows the same 5-section format:

```
0. BRAINSTORM        scan protocol (Task 1) or 3-layer idea drill (Task 2)
1. FLOW              paragraph contract, slot-labeled
2. MUTATION TABLE    switch table: question type → thesis/body/conclusion adaptation
   └── Samples       band 8+ annotated report/essay per mutation (routed, not inline)
3. WORDING BANK      phrase templates anchored to paragraph slot
4. MASTER PROGRESS   binary checklist (all checked = maxed)
```
