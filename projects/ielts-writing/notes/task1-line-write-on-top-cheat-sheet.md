> **TL;DR**: Line graph writing is a 3-step machine: `brainstorm -> structure -> grammar`. First compress the graph, then place the information into a fixed 4-paragraph frame, then choose the cleanest grammar pattern for each sentence.

This note is the execution layer.
The master brainstorm logic already exists in [[projects/ielts-writing/notes/task1-academic]].
This page compresses that logic into a line-graph-specific write-on-top sheet.

---

## 0. BRAINSTORM

```text
2-MIN LINE GRAPH SCAN

Step 1 — Find the extremes
  Highest point / final winner:
    [                    ]
  Lowest point / final loser:
    [                    ]

Step 2 — Name the dominant trend
  Mostly upward / downward / mixed / stable:
    [                    ]

Step 3 — Find one exception
  sharp fall / sudden surge / plateau / crossing / reversal:
    [                    ]

Step 4 — Group the lines
  Body 1 group:
    [                    ]
  Body 2 group:
    [                    ]

Step 5 — Select only support data
  Body 1 numbers:
    [                    ]
  Body 2 numbers:
    [                    ]
```

Brainstorm output must fit this frame:

```text
OVERVIEW 1 = dominant trend
OVERVIEW 2 = strongest contrast / exception
BODY 1    = group 1 with 2-3 strong figures
BODY 2    = group 2 with 2-3 strong figures
```

Compression rule:
- do not describe every year
- do not describe every line separately if they can be grouped
- do not put numbers in the overview

Bad writer:

```text
sees 4 lines -> writes 4 disconnected descriptions
```

Good writer:

```text
sees 4 lines -> compresses them into 2 patterns
```

## 1. STRUCTURE

### Fixed Paragraph Machine

```text
P1 INTRODUCTION
  S1. Paraphrase what the graph shows

P2 OVERVIEW
  S2. Overall, main trend
  S3. Strongest contrast / highest-lowest / exception

P3 BODY 1
  S4. Group 1 topic sentence
  S5. Data sentence 1
  S6. Data sentence 2 / comparison

P4 BODY 2
  S7. Group 2 topic sentence
  S8. Data sentence 3
  S9. Data sentence 4 / final comparison
```

Non-negotiable rules:

```text
S1     = paraphrase only
S2-S3  = overview only, no numbers
S4-S9  = data + comparisons
Task 1 = no conclusion paragraph
```

### Fill-On-Top Template

```text
[P1 INTRO]
S1. The line graph [shows / illustrates / compares] [topic] in [place] from [year] to [year].

[P2 OVERVIEW]
S2. Overall, [main trend].
S3. In addition / Meanwhile / By contrast, [strongest contrast or exception].

[P3 BODY 1]
S4. Looking first at [group 1], [general pattern].
S5. [Line A] [movement pattern] from [x] to [y].
S6. Similarly / By contrast, [Line B] [movement pattern] from [x] to [y].

[P4 BODY 2]
S7. Turning to [group 2], [general pattern].
S8. [Line C] [movement pattern] from [x] to [y].
S9. Meanwhile / In comparison, [Line D] [movement pattern] from [x] to [y].
```

### Structural Variants

Use these when the graph shape changes.

```text
If most lines rise:
  Body 1 = stronger rises
  Body 2 = weaker rises / stable lines

If lines split clearly into rise vs fall:
  Body 1 = rising lines
  Body 2 = falling lines

If one line dominates all others:
  Body 1 = dominant line + nearest competitor
  Body 2 = remaining lower lines

If lines cross each other:
  Body 1 = early-period ranking
  Body 2 = later-period ranking / crossover result

If one line is flat while others move:
  Body 1 = moving lines
  Body 2 = stable line + remaining contrast
```

### Fast Structural Checklist

```text
[ ] Intro paraphrased the prompt
[ ] Overview has no numbers
[ ] Overview names the biggest pattern
[ ] Overview names one contrast / exception
[ ] Body 1 and Body 2 are grouped logically
[ ] Only selected numbers are included
[ ] No useless conclusion added
```

## 2. GRAMMAR PATTERNS

This section is the arsenal.
Do not memorize everything at once.
Use it as a sentence-bank while drilling the structure.

### A. Intro Patterns

```text
The line graph shows [topic] from [year] to [year].
The line graph illustrates changes in [topic] between [year] and [year].
The line graph compares [categories] over the period from [year] to [year].
The given line graph presents data on [topic] in [place] over a [number]-year period.
The graph provides information about how [topic] changed between [year] and [year].
The chart outlines trends in [topic] over the period shown.
```

Intro verbs:

```text
shows
illustrates
compares
presents
provides information about
depicts
outlines
gives data on
```

Time-frame patterns:

```text
from 2000 to 2020
between 2000 and 2020
over the 20-year period
during the period shown
over the timeframe in question
```

### B. Overview Patterns

Overview starters:

```text
Overall,
In general,
On the whole,
It is clear that
It is immediately apparent that
```

Main-trend patterns:

```text
Overall, X increased, while Y declined.
Overall, most categories rose over the period.
Overall, the figure for X showed an upward trend, whereas Y moved in the opposite direction.
Overall, X remained the dominant category throughout the period.
Overall, there was a general increase in X, despite some fluctuations.
Overall, the graph is characterized by growth in X and a decline in Y.
```

Contrast / exception patterns:

```text
In contrast, Y fell substantially.
By comparison, Y remained relatively stable.
Meanwhile, Y was consistently the lowest figure.
The most striking feature is that X overtook Y by the end of the period.
Another notable feature is that Y experienced a sharp drop after 2010.
A further noticeable point is that X started as the lowest figure but ended as the highest.
```

Overview ban:

```text
NO:
  from 20% to 45%
  at 30 units
  peaked at 70 in 2015

YES:
  rose markedly
  remained the highest
  declined sharply
  showed the opposite trend
```

### G. Comparison Patterns

#### Same-direction Comparison

```text
Similarly, X rose from ... to ...
Likewise, X showed a comparable increase.
In the same way, X followed a similar trend.
X also increased over the period.
```

#### Opposite-direction Comparison

```text
By contrast, X fell from ... to ...
In contrast, X declined steadily.
Whereas X increased, Y decreased.
While X rose considerably, Y remained stable.
Unlike X, Y showed a downward trend.
```

#### Ranking Comparison

```text
X was higher than Y throughout the period.
X remained above Y for most of the timeframe.
X started below Y but overtook it in 2015.
By the end of the period, X had surpassed Y.
The figure for X was roughly double that of Y.
X and Y were almost identical in 2010.
```

### H. Core Sentence Architectures

#### Simple Movement Sentence

```text
X rose from 10 to 20.
X fell from 30% to 15%.
X remained stable at around 12 units.
```

#### Movement + Time

```text
X rose steadily from 10 to 20 between 2000 and 2010.
X declined gradually over the first half of the period.
X remained unchanged throughout the timeframe.
```

#### Start -> End Pattern

```text
Starting at 10, X climbed to 30 by 2020.
Beginning at approximately 25%, X fell to just 5% at the end of the period.
From an initial figure of 12 units, X increased steadily to 28.
```

#### Peak Pattern

```text
X peaked at 40 in 2015 before falling slightly.
After rising steadily, X reached a peak of 70 units in 2010.
X hit its highest point in the middle of the period.
```

#### Low Pattern

```text
X bottomed out at 5 in 2008.
The figure for X hit a low of around 10% in 2012.
After a gradual decline, X reached its lowest level at the end of the period.
```

#### Fluctuation Pattern

```text
X fluctuated between 10 and 20 throughout the period.
The figure for X showed considerable fluctuation, despite ending slightly higher than it began.
X varied modestly over the timeframe before stabilizing in the final years.
```

#### Contrast Sentence

```text
While X rose steadily, Y declined sharply.
Whereas X remained stable, Y increased significantly.
X climbed to 40, while Y fell to just 10.
```

#### Comparison Sentence With Nouns

```text
The figure for X was higher than that for Y throughout the period.
The proportion of X was roughly twice that of Y in 2020.
The number of X remained below the figure for Y until 2015.
```

#### Clause-Heavy Band-7 Pattern

```text
Although X started as the least common category, it grew steadily and eventually overtook Y.
While Y experienced a sharp drop after 2010, X continued to rise throughout the remainder of the period.
Having remained stable for several years, X then increased markedly in the final decade.
```

### I. Noun-Based Reporting Patterns

Verb-only writing becomes repetitive. Use noun forms too.

```text
an increase in
a rise in
growth in
a climb in
an upward trend in

a decrease in
a fall in
a decline in
a drop in
a downward trend in

a slight fluctuation in
considerable volatility in
stability in
```

Sentence patterns:

```text
There was a significant increase in X over the period.
X saw a gradual rise between 2000 and 2010.
A sharp decline in Y was recorded after 2015.
The most noticeable change was the increase in X.
```

### J. Body Topic Sentence Patterns

```text
Looking first at X and Y, both categories followed an upward trend.
Turning to X and Y, the two lines moved in opposite directions.
With regard to X and Y, both figures remained relatively stable for most of the period.
As for the remaining categories, their trends were more volatile.
Regarding X, Y, and Z, X remained dominant throughout.
```

### K. Safe Complex-Sentence Patterns

Use a few of these, not all of them.

```text
While X increased steadily, Y declined over the same period.
Although X initially remained stable, it rose sharply after 2010.
After peaking at 40, X fell gradually in the final years.
Having started at a very low level, X increased substantially by the end of the period.
As X climbed to 30, Y fell to just 10.
When the period began, X was higher than Y; however, this pattern later reversed.
```

### L. Ratio / Scale Patterns

```text
twice as high as
three times higher than
half the figure for
slightly above
well below
far higher than
marginally lower than
almost the same as
```

Examples:

```text
By 2020, X was twice as high as Y.
The figure for X was well below that of Y throughout the period.
In the final year, X and Y were almost the same.
```

### M. What To Avoid

```text
"According to the graph, it can be clearly seen that..."
"There was an upward increase."
"It went up upwardly."
"X was more better than Y."
"In a nutshell" / "To conclude" in Task 1
copying the prompt word-for-word
listing every number mechanically
using the same verb 5 times in one report
```

### N. Fast Grammar Selection Map

```text
Need intro?
  -> intro pattern

Need overview?
  -> trend + contrast pattern

Need body sentence for rise?
  -> upward verb + adverb + data

Need body sentence for fall?
  -> downward verb + adverb + data

Need stable line?
  -> stability pattern

Need crossing lines?
  -> overtake / surpass pattern

Need variety?
  -> use noun-based reporting in one sentence
```

---

## Use Protocol

```text
1. Fill brainstorm box first.
2. Lock the 9-sentence structure.
3. For each sentence, choose the cleanest grammar pattern from Section 2.
4. Prioritize clarity over variety.
5. Add variety only after the structure is correct.
```
