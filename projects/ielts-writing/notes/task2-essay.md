> **TL;DR**: One doc for all Task 2 essay types. Same 4-paragraph skeleton, 5 mutations handled via switch table. 250+ words, 40 minutes. PEEL body paragraphs.

## 0. BRAINSTORM — How to Generate Arguments Before You Write

```
3-LAYER IDEA DRILL (5 min — run before writing a single word):

For EACH body paragraph, drill 3 layers in order:

  Layer 1 — CLAIM (state it in one sentence)
    "One reason is that [X]."
    "A significant advantage is that [X]."
    "The main cause is [X]."
    If you cannot state it cleanly, the idea is not ready.

  Layer 2 — MECHANISM (why / how does this work?)
    "This is because [causal process]."
    "This means that [direct consequence]."
    If you cannot explain the mechanism, change the claim.
    ← A claim without a mechanism is Band 5.

  Layer 3 — EXAMPLE (make it concrete)
    ┌──────────────────────────────────────────────────────┐
    │ education  │ technology  │ health      │ economy      │
    │ government │ environment │ workplace   │ transport    │
    │ social     │ urbanisation│ family      │ science      │
    └──────────────────────────────────────────────────────┘
    Pick a domain → "For instance, in [domain], [outcome]."
    You do NOT need real statistics — plausible scenario works.
    AVOID: naming countries or studies you know nothing about.

DRILL OUTPUT (write this BEFORE your intro):
  Body 1: [Claim] — [Mechanism] — [Example domain]
  Body 2: [Claim] — [Mechanism] — [Example domain]
           ↑ use a DIFFERENT domain to avoid repetition

TIME GATE: if drill takes >5 min, your claim is too vague.
  Simplify the claim and drill again.

ROOT CAUSE OF "I DON'T KNOW WHAT TO WRITE":
  Band 5-6 → starts writing before argument is formed
  Band 7+  → drills the mechanism first, writes second
```

## 1. FLOW — Paragraph Contract

```
┌─────────────────────────────────────────────────────────────────┐
│ ¶1  INTRODUCTION  [2-3 sentences]                               │
│     └─ Slot A: paraphrase the question (1 sentence)             │
│        NEVER copy the prompt — swap vocab, change structure      │
│     └─ Slot B: thesis statement (1-2 sentences)                 │
│        ← this is where mutations diverge — see table below      │
│        MUST contain your position + 2 reasons (preview)         │
│                                                                 │
│ ¶2  BODY 1  [5-7 sentences]  ← PEEL structure                  │
│     └─ Slot C: topic sentence (Point — main claim)              │
│     └─ Slot D: explanation (Explain — why/how, 2-3 sentences)   │
│     └─ Slot E: example (Example — concrete, specific)           │
│     └─ Slot F: link back (Link — tie to thesis)                 │
│                                                                 │
│ ¶3  BODY 2  [5-7 sentences]  ← mirror PEEL                     │
│     └─ same slot structure, different reason/argument            │
│     └─ optionally add concession here for band 7+:              │
│        "While it is true that [opposing], [your counter]"       │
│                                                                 │
│ ¶4  CONCLUSION  [2-3 sentences]                                 │
│     └─ Slot G: restate position (rephrase, don't repeat)        │
│     └─ Slot H: recommendation or future implication             │
│        ← this is where mutations diverge — see table below      │
│        NEVER introduce new ideas in the conclusion              │
└─────────────────────────────────────────────────────────────────┘

BODY PARAGRAPH ANATOMY — PEEL in detail:
  ┌────────────────────────────────────────────────────────────┐
  │  P — Point     "One significant reason is that..."         │
  │       ↓                                                    │
  │  E — Explain   "This means that... because..."  (2-3 sen) │
  │       ↓        ← THIS is where band 5 students stop       │
  │  E — Example   "For instance, in [country/context]..."     │
  │       ↓                                                    │
  │  L — Link      "Therefore, it is clear that..."            │
  └────────────────────────────────────────────────────────────┘

  Band 5-6: Point → stop (underdeveloped)
  Band 7+:  Point → Explain → Example → Link (fully extended)

TIME BUDGET (40 min total):
  5 min  — read prompt + identify essay type + plan 2 main ideas
  5 min  — write ¶1 (intro + thesis)
  12 min — write ¶2 (body 1)
  12 min — write ¶3 (body 2)
  3 min  — write ¶4 (conclusion)
  3 min  — proofread for grammar errors
```

### 2b. Body Paragraph Focus — The Switch

```
ESSAY TYPE             │ BODY 1 FOCUS                   │ BODY 2 FOCUS
───────────────────────┼────────────────────────────────┼──────────────────────────
OPINION (fully)        │ Reason 1 supporting position   │ Reason 2 supporting position
OPINION (partial)      │ Acknowledge opposing view      │ Your main position + why
DISCUSSION             │ View you DISAGREE with (first) │ View you AGREE with (last)
ADV-DIS (outweigh)     │ Weaker side                    │ Stronger side (your view)
ADV-DIS (balanced)     │ Advantages                     │ Disadvantages
PROBLEM-SOLUTION       │ Causes / Problems              │ Solutions (linked to causes)
TWO-PART Q             │ Answer to Question 1           │ Answer to Question 2
───────────────────────┴────────────────────────────────┴──────────────────────────

ORDERING RULE: put the side you DISAGREE with first, your position LAST.
               The examiner reads the final body paragraph most recently
               before the conclusion — that's where your strongest case lands.
```

### 2c. Conclusion (Slot G + H) — The Switch

```
ESSAY TYPE             │ SLOT G (restate)               │ SLOT H (recommendation)
───────────────────────┼────────────────────────────────┼──────────────────────────
OPINION                │ "In conclusion, I firmly       │ "[Recommendation or future
                       │ believe that [position]."      │ implication]."
───────────────────────┼────────────────────────────────┼──────────────────────────
DISCUSSION             │ "In conclusion, while both     │ "I believe [your view] is
                       │ sides have valid points..."    │ more compelling because..."
───────────────────────┼────────────────────────────────┼──────────────────────────
ADV-DIS (outweigh)     │ "To sum up, although there     │ "the [stronger side]
                       │ are [weaker side]..."          │ outweighs these concerns."
───────────────────────┼────────────────────────────────┼──────────────────────────
ADV-DIS (balanced)     │ "In conclusion, there are      │ "Individuals should weigh
                       │ merits on both sides."         │ these factors carefully."
───────────────────────┼────────────────────────────────┼──────────────────────────
PROBLEM-SOLUTION       │ "In conclusion, [restate       │ "These solutions could
                       │ the core problem]."            │ significantly reduce..."
───────────────────────┼────────────────────────────────┼──────────────────────────
TWO-PART Q             │ "In conclusion, [answer Q1     │ "[Final reflection linking
                       │ restated] and [Q2 restated]."  │ both answers]."
```

## 3. WORDING BANK — Phrases Anchored to PEEL Slots

### Slot A — Introduction (paraphrase the prompt)

```
STARTERS:
  "It is widely debated whether..."
  "In recent years, [topic] has become a subject of considerable debate."
  "The question of whether [X] is a matter of ongoing discussion."
  "There is a growing trend towards [X], which raises the question of..."
  "[Topic] has attracted significant attention in recent years."

AVOID:
  "In this day and age..."    ← cliched, flags memorized language
  "Since the dawn of time..." ← examiner eye-roll
  "Nowadays..."               ← not wrong, but overused
```

### Slot B — Thesis (see Mutation Table 2a for per-type templates)

### Slot C — Topic Sentence (Point)

```
REASON-BASED:
  "The primary reason for this is that..."
  "One of the most significant factors is..."
  "A key consideration is that..."
  "Perhaps the most compelling argument is that..."

ADVANTAGE/DISADVANTAGE:
  "One notable advantage of [X] is that..."
  "A significant drawback, however, is that..."
  "On the positive side, [X] allows..."

PROBLEM:
  "One of the most pressing issues is..."
  "A major contributing factor is..."

SOLUTION:
  "One effective way to address this is..."
  "A practical solution would be to..."
```

### Slot D — Explanation (Explain)

```
DEVELOPING THE POINT:
  "This is because..."
  "In other words, ..."
  "What this means in practice is that..."
  "The reason for this is that..."
  "This is largely due to the fact that..."

EXTENDING:
  "As a result of this, ..."
  "Consequently, ..."
  "This, in turn, leads to..."
  "Building on this, ..."

CONCESSION (band 7+ move):
  "While it is true that [opposing point], [counter]."
  "Although some may argue that [opposing], [your position]."
  "Despite the fact that [opposing], [counter]."
  "Admittedly, [weakness], but [stronger point]."
```

### Slot E — Example

```
INTRODUCING EXAMPLES:
  "For instance, ..."
  "A clear example of this can be seen in..."
  "To illustrate, ..."
  "This is evident in countries such as..."
  "Research has shown that..."

AVOID:
  "For example, my friend..."     ← too personal/anecdotal for band 7
  Invented statistics              ← examiners spot these
  "According to studies..."        ← fine only if you name the context
```

### Slot F — Link Back

```
  "Therefore, it is clear that [thesis restatement]."
  "This demonstrates that..."
  "For this reason, [position] is justified."
  "This further supports the view that..."
```

### Cohesion Devices — Organized by Function

```
ADDITION (don't overuse — max 1-2 per essay):
  Furthermore  │  Moreover  │  In addition  │  What is more

CONTRAST (rotate these — never use the same one twice):
  However           │  Nevertheless      │  On the other hand
  By contrast       │  Conversely        │  Despite this
  Even so           │  Notwithstanding this

CAUSE / RESULT:
  Consequently      │  As a result       │  Therefore
  Thus              │  For this reason   │  Hence

CONCESSION:
  Although          │  While             │  Even though
  Despite           │  In spite of       │  Regardless of

CLARIFICATION:
  In other words    │  That is to say    │  Specifically
  More precisely    │  To put it differently

⚠ BAND 7+ RULE:
  Do NOT start every sentence with a cohesive device.
  Band 8-9 writers use them ONLY when needed.
  "Also... Also... Also..." = band 5 flag.
  Natural flow > mechanical linking.
  Use referencing (this, these, such) as cohesion too — it's invisible but effective.
```

## 4. MASTER PROGRESS — Binary Checklist

```
SCHEMA:
  [ ] Can identify essay type from question in <30 seconds
  [ ] Can write the 4-paragraph skeleton with correct mutation from memory in <3 min
  [ ] Can produce a thesis for any of the 7 mutations without a template

BODY PARAGRAPHS:
  [ ] Last 3 essays have PEEL structure in BOTH body paragraphs
  [ ] Each body ¶ has ≥5 sentences (not 2-3 underdeveloped ones)
  [ ] Used at least 1 concession phrase ("While...", "Although...") in last 2 essays

WORDING:
  [ ] Zero cliched openings ("In this day and age", "Since time immemorial")
  [ ] Used ≥3 different cohesion categories (addition, contrast, result) per essay
  [ ] No cohesive device appears more than twice in a single essay
  [ ] Paraphrased the question — zero copied phrases from prompt in ¶1

GRAMMAR:
  [ ] Majority of sentences are error-free in last 2 essays
  [ ] Used at least 2 complex structures (conditionals, relative clauses, passive)
  [ ] Self-scored ≥7 on Grammatical Range in last 2 essays

ALL 12 CHECKED = MAXED
```

## BAND 6 → 7 JUMP — What Exactly Changes (Task 2)

```
CRITERION           │ BAND 6 (you now)              │ BAND 7 (target)
─────────────────────┼───────────────────────────────┼─────────────────────────────
Task Response        │ addresses all parts            │ addresses all parts +
                     │ position is present            │ position is CLEAR + CONSISTENT
                     │ ideas relevant but              │ ideas EXTENDED + SUPPORTED
                     │ underdeveloped                 │ with evidence
─────────────────────┼───────────────────────────────┼─────────────────────────────
Coherence & Cohesion │ cohesive devices present       │ cohesion is NATURAL
                     │ but sometimes mechanical       │ each ¶ has clear central topic
                     │ paragraphing may be faulty     │ logical progression throughout
─────────────────────┼───────────────────────────────┼─────────────────────────────
Lexical Resource     │ adequate vocabulary            │ flexible + precise vocabulary
                     │ some errors in word choice     │ COLLOCATIONS over big words
                     │ attempts less common words     │ awareness of style + register
─────────────────────┼───────────────────────────────┼─────────────────────────────
Grammatical Range    │ mix of simple + complex        │ VARIETY of complex structures
                     │ some errors persist            │ frequent ERROR-FREE sentences
                     │ meaning generally clear        │ good control of grammar
```
