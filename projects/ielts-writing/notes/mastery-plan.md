> **TL;DR**: 14-day plan to max both docs. Week 1 = build schema + first reps. Week 2 = max mastery progress on both docs. 6 days buffer before exam.

## The Math

```
2 docs × ~10 master progress criteria each = ~21 total checkboxes
14 days = enough for daily focused reps
6 remaining days (of 20 total) = buffer for mock tests + weakest-link drills
```

---

## WEEK 1 — BUILD (Days 1–7)

Goal: own both schemas cold + write first reps against each mutation.

```
DAY 1 (Apr 9) — Task 1 Schema Day
──────────────────────────────────────────────────────────────────
  Morning:
    □ Read task1-academic.md top to bottom
    □ Copy the 4-paragraph skeleton (¶1-¶4) onto paper by hand
    □ Memorize the mutation table — which graph type → which grouping

  Afternoon:
    □ Write 1 LINE GRAPH report (timed, 20 min)
    □ Self-audit against master progress checklist
    □ Read 1 band 8 model answer (compare your ¶2 overview to theirs)


DAY 2 (Apr 10) — Task 1 Mutations
──────────────────────────────────────────────────────────────────
  □ Write 1 BAR CHART report (20 min)
  □ Write 1 PIE CHART report (20 min)
  □ Self-audit both: did ¶2 have data? (should NOT)
  □ Diff against model: what did they group that you didn't?


DAY 3 (Apr 11) — Task 1 Hard Mutations
──────────────────────────────────────────────────────────────────
  □ Write 1 PROCESS DIAGRAM report (20 min)
     → check: did you use passive voice? sequencing words?
  □ Write 1 TABLE report (20 min)
  □ Self-audit: wording bank usage — did you rotate verbs?


DAY 4 (Apr 12) — Task 2 Schema Day
──────────────────────────────────────────────────────────────────
  Morning:
    □ Read task2-essay.md top to bottom
    □ Copy the PEEL structure onto paper by hand
    □ Memorize the mutation table — which question signal → which thesis

  Afternoon:
    □ Write 1 OPINION essay (fully agree, 40 min)
    □ Self-audit: does each body ¶ have PEEL? count sentences per ¶
    □ Read 1 band 8 model answer


DAY 5 (Apr 13) — Task 2: Opinion + Discussion
──────────────────────────────────────────────────────────────────
  □ Write 1 OPINION essay (partial agree / "to what extent", 40 min)
  □ Write 1 DISCUSSION essay (40 min)
  □ Self-audit: ordering rule — weaker side first, your view last?


DAY 6 (Apr 14) — Task 2: Adv-Dis + Problem-Solution
──────────────────────────────────────────────────────────────────
  □ Write 1 ADVANTAGES-DISADVANTAGES essay (outweigh variant, 40 min)
  □ Write 1 PROBLEM-SOLUTION essay (40 min)
  □ Self-audit: did you link each problem to its solution?


DAY 7 (Apr 15) — Task 2: Two-Part Q + Full Mock
──────────────────────────────────────────────────────────────────
  □ Write 1 TWO-PART QUESTION essay (40 min)
  □ Full 60-min mock: Task 1 (20 min) + Task 2 (40 min)
  □ Self-score both using master progress checklists
  □ Update master progress: check off anything now mastered


WEEK 1 EXIT GATE:
  ✓ Both docs read and internalized
  ✓ At least 5 Task 1 reports written (1 per graph type)
  ✓ At least 5 Task 2 essays written (1 per essay type)
  ✓ Master progress: ≥ 50% of criteria checked on both docs
```

## WEEK 2 — MAX (Days 8–14)

Goal: close every unchecked criterion. Targeted reps, not random practice.

```
DAY 8 (Apr 16) — Task 1 Gap Close
──────────────────────────────────────────────────────────────────
  □ Review task1-academic master progress — which criteria unchecked?
  □ Write 2 reports targeting the weakest mutation (the one you scored lowest on)
  □ Focus drill: if overview was weak → write 5 standalone overviews (no full report)


DAY 9 (Apr 17) — Task 2 Gap Close: Opinion + Discussion
──────────────────────────────────────────────────────────────────
  □ Review task2-essay master progress
  □ Write 1 Opinion essay targeting weakest criterion
  □ Write 1 Discussion essay targeting weakest criterion
  □ If thesis was weak → write 5 standalone thesis statements (no full essay)


DAY 10 (Apr 18) — Task 2 Gap Close: Other Types
──────────────────────────────────────────────────────────────────
  □ Write 1 Adv-Dis essay
  □ Write 1 Problem-Solution essay
  □ Cohesion audit: highlight every cohesive device — did you overuse?


DAY 11 (Apr 19) — Full Mock #2
──────────────────────────────────────────────────────────────────
  □ Full 60-min mock (Task 1 + Task 2, strict timer)
  □ Self-score against master progress
  □ Compare to mock #1 (Day 7) — what improved? what didn't?


DAY 12 (Apr 20) — Error Pattern Day
──────────────────────────────────────────────────────────────────
  □ Review ALL essays written so far
  □ List top 3 recurring grammar errors
  □ List top 3 recurring structural mistakes
  □ Write 1 essay focusing ONLY on eliminating those patterns


DAY 13 (Apr 21) — Vocabulary Precision Day
──────────────────────────────────────────────────────────────────
  □ Review wording banks from both docs
  □ Write 1 Task 1 report using ONLY vocabulary from the bank
  □ Write 1 Task 2 essay — consciously deploy 3 collocations
  □ Check: any "big words" used incorrectly? simplify.


DAY 14 (Apr 22) — Final Mock + Audit
──────────────────────────────────────────────────────────────────
  □ Full 60-min mock (Task 1 + Task 2, strict timer)
  □ Final master progress audit on both docs
  □ If all criteria checked → MAXED
  □ If not → list remaining gaps for buffer days


WEEK 2 EXIT GATE:
  ✓ All master progress criteria checked on BOTH docs
  ✓ At least 3 full mocks completed
  ✓ Top recurring errors identified and patched
  ✓ Wording bank phrases used naturally (not forced)
```

---

## BUFFER DAYS (Days 15–20)

```
Only if needed. Do NOT introduce new material.

  Day 15-16 │ Repeat weakest essay type (1 per day)
  Day 17-18 │ Full mock each day
  Day 19    │ Review all model answers one final time
  Day 20    │ Light review only. No new writing. Trust the schema.
```

---

## Daily Session Contract

```
Every practice session follows this loop:

  ┌─────────────────────────────────────────────────┐
  │  1. PULL the doc (Task 1 or Task 2)             │
  │  2. CHECK master progress — what's unchecked?   │
  │  3. WRITE (timed, strict)                       │
  │  4. SELF-AUDIT against master progress          │
  │  5. READ one model answer (band 8)              │
  │  6. DIFF — 3 specific differences               │
  │  7. UPDATE master progress                      │
  └─────────────────────────────────────────────────┘

  If a criterion has been checked for 3 consecutive sessions → it's locked.
  Focus reps on UNCHECKED criteria only.
```
