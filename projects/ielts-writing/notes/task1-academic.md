> **TL;DR**: One doc for all Task 1 Academic graph types. 4-paragraph contract, 6 mutations handled via a switch table. 150+ words, 20 minutes.

## 0. BRAINSTORM — What to Select Before You Write

```
SCAN PROTOCOL (2 min — run before writing a single word):

Step 1 — FIND THE EXTREMES
  Highest value: what is it? Which category / time period?
  Lowest value:  what is it? Which category / time period?
  → These anchor your overview and body data points.

Step 2 — NAME THE DOMINANT TREND
  Is the overall pattern rising, falling, mixed, or stable?
  Which item is consistently highest or lowest?
  → This becomes Overview Sentence 1 (Slot B).

Step 3 — SPOT THE EXCEPTION
  Is there one category or period that breaks the trend?
  → This becomes Overview Sentence 2 (the contrast).

Step 4 — CHOOSE YOUR BODY GROUPING
  ┌────────────────────────────────────────────────────────┐
  │ Option A — by TIME:     early period vs late period    │
  │ Option B — by CATEGORY: high group vs low group        │
  │ Option C — by TREND:    rising items vs falling items  │
  └────────────────────────────────────────────────────────┘
  Body 1 = first group. Body 2 = second group.
  Pick whichever creates the cleanest natural split.

Step 5 — REJECT EVERYTHING ELSE
  Per body ¶, select 3-4 significant figures only.
  Skip data that does not reinforce the grouping pattern
  — extra numbers create clutter, not score.

ROOT CAUSE OF "I DON'T KNOW WHAT TO WRITE":
  Band 5-6 → tries to describe everything → paralysis
  Band 7+  → selects what matters → writes with purpose
  The scan forces selection BEFORE the pen hits the page.
```

## 1. FLOW — Paragraph Contract

```
┌─────────────────────────────────────────────────────────────────┐
│ ¶1  INTRODUCTION  [1-2 sentences]                               │
│     └─ Slot A: paraphrase the question prompt                   │
│        NEVER copy the question word-for-word                    │
│        USE: "The [chart type] illustrates/compares/shows..."    │
│                                                                 │
│ ¶2  OVERVIEW  [2-3 sentences]                                   │
│     └─ Slot B: "Overall," + 2-3 main features/trends           │
│        NO DATA here — no numbers, no percentages                │
│        This is the MOST IMPORTANT paragraph                     │
│        ← a report without a clear overview CANNOT score >5      │
│           on Task Achievement                                   │
│                                                                 │
│ ¶3  BODY 1  [3-5 sentences]                                     │
│     └─ Slot C: first grouping — describe with SPECIFIC data     │
│        topic sentence → data point → comparison → data point    │
│                                                                 │
│ ¶4  BODY 2  [3-5 sentences]                                     │
│     └─ Slot D: second grouping — describe with SPECIFIC data    │
│        same structure as ¶3, different data grouping             │
│        NO CONCLUSION PARAGRAPH — ¶2 overview serves that role   │
└─────────────────────────────────────────────────────────────────┘

TIME BUDGET (20 min total):
  2 min  — read + identify key features
  3 min  — write ¶1 + ¶2 (intro + overview)
  6 min  — write ¶3 (body 1)
  6 min  — write ¶4 (body 2)
  3 min  — proofread for grammar errors
```

## 2. MUTATION TABLE — How the Schema Adapts Per Graph Type

```
Graph Type      │ Intro verb       │ Overview focus         │ Body grouping strategy
────────────────┼──────────────────┼────────────────────────┼──────────────────────────
LINE GRAPH      │ illustrates      │ overall trend          │ group by TREND DIRECTION
                │ shows changes in │ direction (up/down)    │ (rising lines vs falling)
                │                  │ + start vs end point   │
────────────────┼──────────────────┼────────────────────────┼──────────────────────────
BAR CHART       │ compares         │ highest vs lowest      │ group by CATEGORY or
                │ shows            │ overall pattern        │ TIME PERIOD
────────────────┼──────────────────┼────────────────────────┼──────────────────────────
PIE CHART       │ illustrates the  │ largest vs smallest    │ group by SIZE
                │ proportion of    │ segment                │ (dominant vs minor)
────────────────┼──────────────────┼────────────────────────┼──────────────────────────
TABLE           │ compares         │ highest vs lowest      │ group by ROW or COLUMN
                │ presents data on │ notable patterns       │ (whichever reveals pattern)
────────────────┼──────────────────┼────────────────────────┼──────────────────────────
PROCESS DIAGRAM │ illustrates the  │ number of stages       │ FOLLOW THE FLOW
                │ process by which │ + start point          │ split at natural midpoint
                │                  │ + end product          │ USE PASSIVE VOICE
────────────────┼──────────────────┼────────────────────────┼──────────────────────────
MAP             │ illustrates      │ overall scale of       │ group by AREA or by
                │ changes to       │ change (major vs       │ TIME PERIOD (before/after)
                │                  │ minor transformation)  │ USE: "was replaced by"
```

### Mutation-Specific Language

```
PROCESS DIAGRAM — must use:
  ├── Passive voice: "The water is filtered" NOT "They filter the water"
  ├── Sequencing: initially → subsequently → following this → finally
  └── Stage counting: "The process consists of [X] stages"

MAP — must use:
  ├── Spatial: to the north/south/east/west of, adjacent to, opposite
  ├── Change: was replaced by, was converted into, was demolished
  └── Time frame: "By 2020, the area had been transformed into..."

PIE CHART — must use:
  ├── Proportion: accounted for, comprised, represented, made up
  ├── Fractions: roughly one-third, just under a half, approximately a quarter
  └── Dominance: the largest share, the majority of, a significant proportion
```

## 3. WORDING BANK — Phrases Anchored to Paragraph Slot

### Slot A — Introduction (paraphrase the prompt)

```
"The [line graph / bar chart / pie chart / table] [VERB] [TOPIC] in [PLACE] [TIME]."

VERBS (rotate, never repeat across essays):
  illustrates    │ shows           │ compares
  presents       │ provides data on│ gives information about
  depicts        │ displays        │ outlines

EXAMPLES:
  "The bar chart compares the number of students enrolled in three universities between 2010 and 2020."
  "The line graph illustrates changes in electricity consumption in four countries over a 20-year period."
```

### Slot B — Overview (main features, NO data)

```
STARTERS (always begin with one of these):
  "Overall,"                          ← safest, always works
  "In general,"                       ← alternative
  "It is clear that"                  ← for strong patterns
  "It is immediately apparent that"   ← for obvious contrasts

PATTERN TEMPLATES:
  "[X] experienced the most significant [rise/change], while [Y] remained relatively [stable/low]."
  "The most striking feature is that [TREND], whereas [CONTRAST]."
  "[X] was consistently the [highest/lowest] throughout the period."
```

### Slot C/D — Body Paragraphs (data description)

```
TREND UP (verb + adverb):
  ┌──────────────────────────────────────────────────────────┐
  │ INTENSITY     │ VERB              │ ADVERB               │
  │───────────────┼───────────────────┼──────────────────────│
  │ Slight        │ rose / increased  │ slightly / marginally│
  │ Moderate      │ climbed / grew    │ steadily / gradually │
  │ Sharp         │ surged / soared   │ sharply / rapidly    │
  │ Dramatic      │ skyrocketed       │ dramatically         │
  └──────────────────────────────────────────────────────────┘

TREND DOWN (verb + adverb):
  ┌──────────────────────────────────────────────────────────┐
  │ INTENSITY     │ VERB              │ ADVERB               │
  │───────────────┼───────────────────┼──────────────────────│
  │ Slight        │ dipped / fell     │ slightly / marginally│
  │ Moderate      │ declined / dropped│ steadily / gradually │
  │ Sharp         │ plummeted/plunged │ sharply / rapidly    │
  │ Dramatic      │ collapsed         │ dramatically         │
  └──────────────────────────────────────────────────────────┘

STABILITY:
  remained stable │ leveled off │ plateaued │ stayed constant
  "remained unchanged at [X]" │ "stabilized at around [X]"

FLUCTUATION:
  "fluctuated between [X] and [Y]"
  "experienced considerable fluctuation"

PEAKS AND TROUGHS:
  "reached a peak of [X] in [YEAR]"    │ "peaked at [X]"
  "hit a low of [X]"                   │ "bottomed out at [X]"

COMPARISONS (within body paragraphs):
  while          │ whereas       │ compared to
  in contrast    │ by contrast   │ on the other hand
  similarly      │ likewise      │ in the same way

TIME PHRASES:
  "between [X] and [Y]"     │ "from [X] to [Y]"
  "over the period"          │ "during the period"
  "over the following decade" │ "in the subsequent years"

DATA INTEGRATION (how to weave numbers in):
  "[Subject] rose sharply from [X] in [YEAR] to [Y] in [YEAR]."
  "Starting at [X], [subject] experienced a steady increase, reaching [Y] by [YEAR]."
  "The figure for [X] stood at [NUMBER], compared to [Y] at [NUMBER]."
```

## 4. MASTER PROGRESS — Binary Checklist

```
SCHEMA:
  [ ] Can write the 4-paragraph skeleton (¶1-¶4 with slot labels) from memory in <2 min
  [ ] Can identify the correct mutation (graph type → grouping strategy) within 30 seconds

OVERVIEW:
  [ ] Last 3 practice reports have a clear overview with NO data in ¶2
  [ ] Overview names 2-3 key features that match what the examiner would select

WORDING:
  [ ] Used ≥3 different trend verbs (not just "increased/decreased") in last 2 reports
  [ ] Used ≥2 comparison phrases (while, whereas, compared to) per report
  [ ] Zero instances of copied question language in ¶1 — always paraphrased

GRAMMAR:
  [ ] Process diagram: used passive voice for ≥80% of verbs
  [ ] Line/bar: correctly used past tense for past data, present for current
  [ ] Self-scored ≥7 on Coherence & Cohesion in last 2 reports (natural flow, not mechanical)

ALL 9 CHECKED = MAXED
```

                     │ "Also... Also... Also..."      │ logical progression is smooth

                     │ some unnatural choices         │ collocations over big words
                     │                                │ rare unnatural choices

                     │ but with errors                │ complex structures accurate
                     │ meaning generally clear        │ accuracy > complexity attempts
