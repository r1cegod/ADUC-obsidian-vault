## Definition

Escape means the next move is chosen from leverage instead of constraint.

```text
constraint-driven life
  -> low room density
  -> weak external validation
  -> no money optionality
  -> geography decides options

leverage-driven life
  -> proof creates trust
  -> trust opens rooms
  -> rooms create opportunities
  -> money creates optionality
  -> geography becomes a choice
```

Do not reduce escape to Singapore. Singapore is a geography gate. The route must build enough proof, network, and optionality to make that gate realistic.

## Four Escapes

```text
1. Identity escape
   -> stop being passive / average-room-shaped
   -> become builder by evidence

2. Room escape
   -> leave low-density social environment
   -> enter builder/reviewer/customer rooms

3. Money escape
   -> create value people can pay for
   -> earn enough optionality to reduce dependence

4. Geography escape
   -> move toward Singapore or another leverage-dense environment
   -> only after proof/options can carry the move
```

Current status:

```text
identity escape: started
room escape: weak
money escape: early
geography escape: not yet
```

## Anti-Traps

### Trap 1 - Private Builder Cave

```text
builds impressive private systems
  -> nobody sees them
  -> no rooms open
  -> frustration becomes proof of being unseen
```

Counter:

```text
every major build must eventually produce a proof artifact
```

### Trap 2 - Over-OSing

```text
maps the life system forever
  -> no shipped leverage artifact
```

Counter:

```text
OS pages must choose physical moves
```

### Trap 3 - Singapore Fantasy Compression

```text
Singapore = freedom
  -> unclear actual route
  -> anxiety loops
```

Counter:

```text
separate geography from proof, money, legal route, and room density
```

### Trap 4 - Shallow-Room Negotiation

```text
tries to be understood by low-density room
  -> burns energy
  -> blames self or world
```

Counter:

```text
use proof to enter denser rooms
```

### Trap 5 - Feature Addiction

```text
adds Raven/PathFinder features
  -> avoids packaging proof
```

Counter:

```text
proof packaging is part of the build, not decoration
```
